# Budgeting Website (Spreadsheet-Inspired)

A full-stack budgeting web application inspired by Excel-based personal finance systems.

## Tech Stack

- Frontend: HTML, CSS, JavaScript, React
- Backend: Node.js (planned)
- Database: SQL (planned)
- Version Control: Git (GitLab)

## Features (Planned)

- Recurring income and expense reference tables
- Transaction-based source income and expenses
- Auto-calculated monthly budget views
- Spreadsheet-like UI with reactive updates
- Calendar-based monthly breakdowns

## Purpose

This project demonstrates front-end, back-end, data modeling, and financial logic skills for a
computer science portfolio.
