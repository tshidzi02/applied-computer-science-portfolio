This project is a structured, classroom-ready collection of
Python and Java lessons designed for high-school learners.

## What This Project Demonstrates
- Mastery of CS fundamentals
- Teaching readiness
- Clear explanations and progression
- Terminal-based execution

## How to Use
- Start with Python basics
- Progress topic by topic
- Run files using PowerShell
- Extend lessons with challenges
